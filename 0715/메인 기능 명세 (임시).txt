# 주 기능

### 공동 작업

- **팀 모집**
    - 창작물 시놉시스 게시
    - 카테고리 지정
    - 카테고리와 해당하는 작가 추천 (유료)
    - 개인 메시지, 화상회의, 창작자 룸 개설 (WebRTC)
    - 작가별 권한 부여
- **작업**
    - 개인별 작업 공간 할당 (브런치 분리)
    - 블록별로 글 작성, 저장
    - 저장 시 브런치 별로 버전 기록
    - 메인에 병합 시 버전 추가
    - 충돌 발생 시 충돌 현황 확인
    - 충돌 (Diff) 조정
    - 댓글 및 메모: 특정 부분에 댓글 메모 추가 기능 (구글 Docs)
    - 자동 저장: 편집 중인 문서를 자동으로 저장

# 추가 기능

- 캐릭터 데이터베이스: 각 캐릭터의 정보, 배경, 특성 등을 저장하고 관리
- 타임라인: 소설의 주요 이벤트를 시각적으로 표시하는 타임라인
- 링크 및 참조: 관련 링크와 문서를 문서 내에 삽입하고 참조

- **메인 작가**
    - 권한을 가지고 미리 스켈레톤 작성
- **출판 기능**
    - eBook or PDF 변환
    - 출판 플랫폼 연동
- **멀티미디어 삽입 시 저작권 관리**
- **자동 완성 및 제안**
    - 문서 작성 시 AI 기반의 자동 완성 및 문구 제안 (코파일럿과 유사)
- **할당 및 알림**
- **세션 만료 기한**
- **교육 자료 및 튜토리얼**
    - 사용 난이도 낮추기
    - 작가 글쓰기 교육 자료 (AI 연동 및 실제 작가 설명회)

### 버전 관리

- 자동 버전 생성: 일정 시간 간격 또는 주요 변경사항이 발생할 때 자동으로 새로운 버전 생성
- 버전 비교 도구: 두 버전 간의 변경사항을 시각적으로 비교할 수 있는 도구 제공
- 버전 복원: 이전 버전으로 쉽게 복원할 수 있는 기능
- 버전 태그: 특정 버전을 태그하고, 주요 변경사항을 쉽게 식별할 수 있도록 메모 추가
- 버전 설명: 각 버전에 대한 설명을 추가하여 변경 내역을 명확하게 기록

### 작업 관리

- 작업 상태 표시: 각 작업의 상태(작성 중, 리뷰 중, 완료 등)를 제공
- 공동 작업 공간: 작가들이 공동으로 작업할 수 있는 전용 작업 공간 제공
- 작가별 기여도 추적: 각 작가의 기여도를 추적하고 시각화

### 출판 및 배포 준비

- 프리뷰 모드: 최종 출판 전 문서의 미리보기를 제공
- 출판 일자 설정: 출판 일정을 설정하고 자동으로 출판되도록 예약
- 출판 형식 지원: 다양한 출판 형식(eBook, PDF, HTML 등)으로 변환하여 저장
- 출판 플랫폼 연동: 다양한 출판 플랫폼(Amazon Kindle, Apple Books 등)과 연동하여 쉽게 배포

### 콘텐츠 관리 및 탐색

- 콘텐츠 카테고리: 소설, 단편, 시 등 다양한 콘텐츠 유형을 카테고리로 분류
- 태그 및 키워드: 문서에 태그와 키워드를 추가하여 검색과 분류를 용이하게
- 북마크 기능: 특정 페이지나 섹션을 북마크하여 빠르게 접근
- 목차 생성: 자동으로 목차를 생성하여 문서 내 탐색을 쉽게

### 사용자 및 권한 관리

- 협업 초대: 다른 작가를 초대하여 프로젝트에 참여
- 역할 기반 권한 설정: 각 작가에게 역할에 따른 권한(읽기, 편집, 관리)을 부여
- 접근 로그: 누가 언제 문서에 접근하고 편집했는지 기록

### 커뮤니케이션 도구

- 작가 포럼: 작가들이 의견을 나눌 수 있는 포럼 제공
- 토론 스레드: 특정 주제에 대해 토론할 수 있는 스레드 생성
- 피드백 및 제안: 독자나 편집자로부터 피드백과 제안을 받을 수 있는 기능

## Frontend 기능 요구사항 명세

### 상단 네브바

- 장르별 추천
- 릴레이소설
- 전체작품
- 공동창작 인원모집란
- 대여권
- 작품 등록

### 메인 본문

- **투데이 베스트**
- **장르별 베스트**
- **오늘의 릴레이** (릴레이소설)
- 최근에 작성한 소설 연재 가능한 공간으로 이동하는 버튼
- 최근에 본 작품
- 제품 설명

### 인원모집란

- 사용자가 쓰고 싶은 소설의 개요나 초안 등을 올리는 공간
- 모집 기간과 인원을 정하고 시작 버튼을 누르면 소설 연재 공간이 생성됨
- **마이페이지** 또는 **내 서고**에서 현재 연재 중인 작품 확인 가능
- 작품 페이지에서 버전 관리 기능 포함
- 팀장의 승인이 필요한 참여 가능 형식
- 리뷰 등에서 스포일러와 댓글 욕설 방지 기능
- 작업 공간 접근 시 브랜치 선택 후 공간 열기
- 별도의 공동 작업 공간 생성 후 다른 버전으로 정리
- **그림판**과 같은 기능을 넣어 그림을 실시간으로 공유하고 다운로드 가능하게 설정 (DB에 저장하지 않음)
- 실시간 음성 공유 기능
- 자동 저장 기능 추가 고려
- 반응형 디자인 적용 필수 고려
- 회차별 목록 표시 기능 및 회차 감상 기능 제공
- 읽는 페이지에서는 목록, 댓글, 이전/다음 회차 이동, 공유 기능 (카카오 공유 등) 제공
- 후원 기능
- 뷰어 세부 설정 기능 추가 예정
- 작품 페이지 하단에 회차 목록 및 채팅 방 탭 제공 (채팅방 탭으로 회차 기능 이동 가능)
- 작품 별 장르와 태그 표시

## 작품 관리 페이지

### 작품 등록 페이지 (새작품 등록)

- 연재 시작 시 공동 작성 여부 선택
- 공동 작성 시 멤버 초대 가능
- 새로운 모집 글 생성 가능
- 장르별로 다음 항목 제공:
    - 작품 제목 입력
    - 작품 장르 선택
    - 작품 소개 및 시놉시스 작성
    - 표지 이미지 업로드
    - 태그 추가
    - 출판 설정 (공개/비공개, 무료/유료 등)
    - 최종 검토 및 등록

### 지류책 펀딩 신청

- 완결된 작품에 좋아요 기준으로 펀딩 신청 가능
- 약관 동의 후 페이지 생성
- 결제 시스템 구축 (백)
- 수수료 체계 고려
- 최소 금액 설정: 책값 + 추가 금액

### 작성 목록

- 작성 중인 목록을 클릭하여 방에 접속

## 결제 시스템

- 쿠키 충전을 통한 유료 연재 시스템
